🔄 Lekcja 5: Wszystkie typy pętli

JavaScript - while, do...while, for...of, forEach

📌 Przypomnienie: Pętla FOR (Lekcja 4)

To już znamy! Pętla FOR używana gdy znamy liczbę iteracji:

Składnia pętli FOR:
for (inicjalizacja; warunek; krok) {
    // kod do wykonania
}
Przykład:
for (let i = 0; i < 5; i++) {
    console.log("Iteracja: " + i);
}
// Wynik: 0, 1, 2, 3, 4
✅ Kiedy używać FOR:
  • Znasz dokładną liczbę iteracji
  • Potrzebujesz indeksu (i)
  • Iterujesz po tablicy z modyfikacją

🔄 Pętla WHILE - sprawdź POTEM wykonaj

WHILE wykonuje kod dopóki warunek jest prawdziwy. Sprawdza warunek PRZED każdą iteracją.

Składnia:
while (warunek) {
    // kod do wykonania
    // PAMIĘTAJ: zmień warunek żeby pętla się skończyła!
}
Przykład 1 - Podstawowe liczenie:
let i = 0;
while (i < 5) {
    console.log(i);
    i++;  // WAŻNE: zwiększ licznik!
}
// Wynik: 0, 1, 2, 3, 4
Przykład 2 - Losowanie dopóki nie trafisz:
let wylosowana = 0;
let proby = 0;

while (wylosowana !== 6) {
    wylosowana = Math.floor(Math.random() * 6) + 1;
    proby++;
    console.log("Próba " + proby + ": " + wylosowana);
}

console.log("Wylosowano 6 po " + proby + " próbach!");
Przykład 3 - Gra: Zgadywanie liczby:
let tajnaLiczba = 42;
let odpowiedz = 0;
let proby = 0;

while (odpowiedz !== tajnaLiczba) {
    odpowiedz = parseInt(prompt("Zgadnij liczbę (1-100):"));
    proby++;
    
    if (odpowiedz < tajnaLiczba) {
        alert("Za mało!");
    } else if (odpowiedz > tajnaLiczba) {
        alert("Za dużo!");
    }
}

alert("Brawo! Zgadłeś po " + proby + " próbach!");
⚠️ UWAGA: Nieskończona pętla! 
// ❌ BŁĄD - pętla nigdy się nie skończy!
let i = 0;
while (i < 5) {
    console.log(i);
    // Brak i++! Pętla w nieskończoność!
}

// ✅ POPRAWNIE:
let i = 0;
while (i < 5) {
    console.log(i);
    i++;  // Zwiększamy licznik!
}
✅ Kiedy używać WHILE:
  • Nie wiesz ile będzie iteracji
  • Warunek jest złożony
  • Czekasz aż coś się stanie (np. losowanie)
  • Walidacja danych wejściowych
  • Gry i symulacje
✏️ Ćwiczenie 1: Napisz pętlę while która liczy od 10 do 1 (odliczanie).

🔁 Pętla DO...WHILE - wykonaj POTEM sprawdź

DO...WHILE wykonuje kod MINIMUM RAZ, a dopiero potem sprawdza warunek.

Składnia:
do {
    // kod do wykonania (ZAWSZE minimum raz)
} while (warunek);
Przykład 1 - Różnica między while a do...while:
// WHILE - może się w ogóle nie wykonać
let i = 10;
while (i < 5) {
    console.log("To się NIE wyświetli!");
}

// DO...WHILE - wykona się MINIMUM RAZ
let j = 10;
do {
    console.log("To się wyświetli MIMO ŻE j >= 5!");
} while (j < 5);
Przykład 2 - Menu programu:
let wybor;

do {
    console.log("=== MENU ===");
    console.log("1. Dodaj zadanie");
    console.log("2. Pokaż zadania");
    console.log("3. Usuń zadanie");
    console.log("0. Wyjście");
    
    wybor = parseInt(prompt("Wybierz opcję:"));
    
    if (wybor === 1) {
        console.log("Dodaję zadanie...");
    } else if (wybor === 2) {
        console.log("Pokazuję zadania...");
    } else if (wybor === 3) {
        console.log("Usuwam zadanie...");
    }
    
} while (wybor !== 0);

console.log("Do widzenia!");
Przykład 3 - Walidacja z minimum jednym pytaniem:
let liczba;

do {
    liczba = parseFloat(prompt("Podaj liczbę dodatnią:"));
    
    if (liczba <= 0) {
        alert("To nie jest liczba dodatnia! Spróbuj ponownie.");
    }
} while (liczba <= 0);

console.log("Podano poprawną liczbę: " + liczba);
✅ Kiedy używać DO...WHILE:
  • Kod musi wykonać się MINIMUM RAZ
  • Menu, które zawsze się pokazuje
  • Walidacja - zapytaj przynajmniej raz
  • Powtarzanie akcji aż użytkownik potwierdzi
✏️ Ćwiczenie 2: Napisz do...while które pyta o hasło dopóki nie będzie miało min 8 znaków.

📦 Pętla FOR...OF - iteruj po wartościach

FOR...OF iteruje bezpośrednio po wartościach tablicy. Nie potrzebujesz indeksu!

Składnia:
for (let element of tablica) {
    // element to kolejna wartość z tablicy
}
Przykład 1 - Porównanie FOR vs FOR...OF:
let owoce = ["jabłko", "banan", "gruszka"];

// Stary sposób - FOR z indeksem
for (let i = 0; i < owoce.length; i++) {
    console.log(owoce[i]);
}

// Nowy sposób - FOR...OF (prostszy!)
for (let owoc of owoce) {
    console.log(owoc);
}

// Oba dają ten sam wynik:
// jabłko
// banan
// gruszka
Przykład 2 - Obliczanie sumy:
let ceny = [10, 25, 30, 15];
let suma = 0;

for (let cena of ceny) {
    suma += cena;
}

console.log("Suma: " + suma + " zł"); // 80 zł
Przykład 3 - Filtrowanie (wypisz tylko oceny >= 4):
let oceny = [5, 3, 4, 2, 6, 4];

console.log("Dobre oceny:");
for (let ocena of oceny) {
    if (ocena >= 4) {
        console.log(ocena);
    }
}
// Wynik: 5, 4, 6, 4

🔍 Kiedy FOR a kiedy FOR...OF?

✅ Użyj FOR jeśli:
  • Potrzebujesz indeksu (i)
  • Modyfikujesz tablicę
  • Generujesz HTML z numerami
  • Przeskakujesz elementy (i += 2)
// Przykład - potrzebujemy numeru
for (let i = 0; i < uczniowie.length; i++) {
    console.log((i+1) + ". " + uczniowie[i]);
}
✅ Użyj FOR...OF jeśli:
  • Iterujesz po wartościach
  • NIE potrzebujesz indeksu
  • Prostsze, czytelniejsze
  • NIE modyfikujesz tablicy
// Przykład - tylko wartości
for (let uczen of uczniowie) {
    console.log("Witaj " + uczen);
}
⚠️ FOR...OF nie modyfikuje tablicy: 
let liczby = [1, 2, 3];

// ❌ To NIE zmieni oryginalnej tablicy!
for (let liczba of liczby) {
    liczba = liczba * 2;
}
console.log(liczby); // [1, 2, 3] - bez zmian!

// ✅ Użyj FOR z indeksem:
for (let i = 0; i < liczby.length; i++) {
    liczby[i] = liczby[i] * 2;
}
console.log(liczby); // [2, 4, 6] - zmienione!
✏️ Ćwiczenie 3: Użyj for...of aby wypisać wszystkie oceny i obliczyć średnią.

🎯 Metoda forEach() - funkcja dla każdego elementu

forEach() to metoda tablicy, która wykonuje funkcję dla każdego elementu. Daje dostęp do wartości, indeksu i całej tablicy.

Składnia:
tablica.forEach(function(element, indeks, tablica) {
    // element - wartość
    // indeks - pozycja (opcjonalnie)
    // tablica - cała tablica (opcjonalnie)
});
Przykład 1 - Podstawowe użycie:
let owoce = ["jabłko", "banan", "gruszka"];

owoce.forEach(function(owoc) {
    console.log("Lubię " + owoc);
});

// Wynik:
// Lubię jabłko
// Lubię banan
// Lubię gruszka
Przykład 2 - Z indeksem:
let uczniowie = ["Anna", "Bartek", "Czarek"];

uczniowie.forEach(function(uczen, indeks) {
    console.log((indeks + 1) + ". " + uczen);
});

// Wynik:
// 1. Anna
// 2. Bartek
// 3. Czarek
Przykład 3 - Generowanie HTML:
let produkty = ["Laptop", "Mysz", "Klawiatura"];
let html = "";

produkty.forEach(function(produkt, indeks) {
    html += "<li>Produkt " + (indeks + 1) + ": " + produkt + "</li>";
});

document.getElementById('lista').innerHTML = html;
Przykład 4 - Arrow function (nowoczesna składnia):
let liczby = [1, 2, 3, 4, 5];

// Tradycyjna funkcja
liczby.forEach(function(liczba) {
    console.log(liczba * 2);
});

// Arrow function (krótsza)
liczby.forEach(liczba => {
    console.log(liczba * 2);
});

// Jeszcze krócej (jedna linia)
liczby.forEach(liczba => console.log(liczba * 2));
✅ Kiedy używać forEach:
  • Wykonujesz akcję dla KAŻDEGO elementu
  • Potrzebujesz wartości i indeksu
  • Nowoczesny, czytelny kod
  • Generowanie HTML z numeracją
⚠️ forEach vs for...of - różnice:
  • forEach - NIE można przerwać (break/continue nie działa)
  • for...of - MOŻNA użyć break i continue
// ❌ BREAK nie działa w forEach
let liczby = [1, 2, 3, 4, 5];
liczby.forEach(liczba => {
    if (liczba === 3) {
        break; // BŁĄD! Nie można przerwać forEach
    }
    console.log(liczba);
});

// ✅ BREAK działa w for...of
for (let liczba of liczby) {
    if (liczba === 3) {
        break; // OK! Przerwie pętlę
    }
    console.log(liczba);
}
✏️ Ćwiczenie 4: Użyj forEach aby wyświetlić listę zadań z numerami i przyciskami "Usuń".

⚠️ Najczęstsze błędy uczniów

BŁĄD 1: Nieskończona pętla while

// ❌ BŁĄD
let i = 0;
while (i < 5) {
    console.log(i);
    // Brak i++ - pętla w nieskończoność!
}

// ✅ POPRAWNIE
let i = 0;
while (i < 5) {
    console.log(i);
    i++;  // Pamiętaj zwiększyć!
}

BŁĄD 2: Próba modyfikacji w for...of

// ❌ BŁĄD - to nie zmieni tablicy!
let liczby = [1, 2, 3];
for (let liczba of liczby) {
    liczba = liczba * 2;  // Zmienia tylko lokalną zmienną!
}

// ✅ POPRAWNIE - użyj FOR z indeksem
for (let i = 0; i < liczby.length; i++) {
    liczby[i] = liczby[i] * 2;
}

BŁĄD 3: Mylenie while z do...while

// WHILE - może się NIE wykonać
let x = 10;
while (x < 5) {
    console.log("Nie wyświetli się!");
}

// DO...WHILE - wykona się MINIMUM RAZ
let y = 10;
do {
    console.log("Wyświetli się mimo że y >= 5!");
} while (y < 5);

BŁĄD 4: Break w forEach (nie działa!)

// ❌ BŁĄD - break nie działa w forEach
[1,2,3].forEach(x => {
    if (x === 2) break;  // BŁĄD!
});

// ✅ POPRAWNIE - użyj for...of
for (let x of [1,2,3]) {
    if (x === 2) break;  // OK!
}

📝 Podsumowanie

Czego się nauczyłeś:

  • ✅ Pętla WHILE - sprawdza przed, używaj gdy nie wiesz ile iteracji
  • ✅ Pętla DO...WHILE - wykonuje minimum raz, idealna do menu
  • ✅ Pętla FOR...OF - iteruje po wartościach, prostsza niż FOR
  • ✅ Metoda forEach - nowoczesna, z dostępem do indeksu
  • ✅ Kiedy której pętli użyć - drzewo decyzyjne
